# File Explorer Application (C++)
A console-based File Explorer written in C++ that interacts with the Linux OS to perform basic file and directory operations.

## Features
- List files and directories
- Navigate between folders
- Copy, move, and delete files
- Search files recursively
- View file permissions

## Tech Stack
- Language: C++17  
- Platform: Linux  
- Headers: `<dirent.h>`, `<sys/stat.h>`, `<unistd.h>`, `<fstream>`

## Run Instructions
```bash
make
./file_explorer
